# Personal Portfolio Website

## Overview

A responsive personal portfolio website developed using HTML, CSS, JavaScript, and Node.js. This project showcases my profile, skills, and contact information in a simple and user-friendly interface.

## Features

* Personal profile section
* About Me section
* Skills showcase
* Contact form with JavaScript validation
* Responsive design
* Basic Node.js backend server

## Technologies Used

* HTML5
* CSS3
* JavaScript
* Node.js
* Express.js

## Project Structure

```text
Portfolio/
├── index.html
├── style.css
├── script.js
├── server.js
└── profile.jpg.jpeg
```

## How to Run

1. Install dependencies:

```bash
npm install express
```

2. Start the server:

```bash
node server.js
```

3. Open `index.html` in your browser.
